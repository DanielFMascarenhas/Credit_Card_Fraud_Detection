# Credit_Card_Fraud_Detection

Created a classification Model for a training data set which was very imbalanced. Also all parameters are PCA's (considering sensitive financial info).

Data: https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud/data

Model Accuracy: 99.956 % . But Since, fraudulant classfications were too low for the given dataset, hence had to compute AUC score too. That was 88.47%
